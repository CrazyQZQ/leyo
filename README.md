# 工程简介
基于spring cloud alibaba的微服务商城项目，旨在学习熟练微服务开发

[移动端项目地址](https://github.com/CrazyQZQ/leyo_ui)
[后台管理前端项目地址](https://github.com/CrazyQZQ/leyo_admin_ui)

# 项目结构

├─account-server 账户服务 

├─common 公共模块

│  ├─common-core
                         
│  ├─common-datasource
                        
│  ├─common-es
                         
│  ├─common-log
                        
│  ├─common-rabbit
                      
│  ├─common-redis
                        
│  └─common-system
                          
├─es-server es服务
                           
├─gateway-server 网关服务
   
├─oauth2-resouce-demo oauth2资源测试服务
                        
├─oauth2-server oauth2服务
                       
├─order-server 订单服务
                       
├─product-server 商品服务
  
└─system-server 系统服务
